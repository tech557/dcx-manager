# Superseded sprint set — flat requirements-logging model (pre-2026-06-29 redesign)

These eight files (RS-R0 … RS-R6) are the **original** requirements-system sprint set. They reached an
audit verdict of READY (`../../audit/2026-06-29-codex-ready.md`) but were **superseded on 2026-06-29**
when the PO expanded the product direction from a *governed requirements store* to a **living
product-intelligence and governance graph** (intent → requirement → behavior → responsibilities →
expected/actual manifestations → verification evidence, bidirectional, self-governing).

They are kept as record (archive-not-destroy). **Do not execute them.** The active drafted sprint set is
in `../` (RS-R0a … RS-R11). The mapping from old → new and the full conflict/supersession list is in
`../../CHANGE-SUMMARY.md`. The audit history in `../../audit/` refers to line numbers in these old files.
